# fabric-h5-Vue3-Demo

前端基于js的一个强大而简单的 Javascript HTML5 canvas库，该仓库为使用示例代码仓库
官网地址：`http://fabricjs.com`

## 安装fabric

H5中使用：下载并引入fabric.min.js文件即可。

Vue中使用：`npm i fabric` 安装即可

## 使用方法

H5中的使用方法见：**H5-Demo** 文件夹

Vue3中的使用方法见：**Vue3-Demo** 文件夹
